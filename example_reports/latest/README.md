# Latest Example Delta Report

This folder always contains the **most recent** example report generated from OpenClaw 7-day delta.

- **LATEST_delta_report.html** → Interactive HTML dashboard (open in browser)
- **LATEST_delta_summary.xlsx** → Full Excel report with commits and file changes

The GitHub Actions workflow automatically updates these files on every push to main and daily at midnight UTC.

You can review the latest report anytime right here in the repo!

**Next update**: Trigger the workflow manually from the "Actions" tab if you want an instant refresh.
